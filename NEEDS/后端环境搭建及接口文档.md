## 一、环境搭建（A 自己跑后端）

### 1. 克隆仓库

```
git clone https://github.com/silentmonkeys/software2026.git
cd software2026
git checkout dev-B
```

### 2. 安装 Python 依赖

```
cd backend
pip install -r requirements.txt
```

> 需要 Python 3.10+，没装的话先去 python.org 下载

### 3. 配环境变量

复制 `.env.example` 为 `.env`：

```
cp .env.example .env
```

打开 `.env`，把 `DASHSCOPE_API_KEY=sk-xxx` 换成你自己的 Key（去 https://bailian.console.aliyun.com 免费申请一个，新用户有免费额度）

### 4. 启动后端

```
uvicorn app.main:app --reload
```

启动后浏览器打开：`http://127.0.0.1:8000/docs`

## 二、接口总览（12 个全部上线）

鉴权方式：JWT Bearer Token（**必须先登录，后续所有接口必须带 Authorization 头**）

### 认证模块 /api/auth

| 方法 | 路径 | 说明 | 请求体 | 返回 |
|---|---|---|---|---|
| POST | `/api/auth/register` | 注册 | `{"username":"", "password":""}` | `{"access_token":"", "role":""}` |
| POST | `/api/auth/login` | 登录 | `{"username":"", "password":""}` | `{"access_token":"", "role":""}` |

> 登录后拿到的 access_token 存 localStorage，后续所有请求加 `Authorization: Bearer <token>`

### 知识库模块 /api/kb（需鉴权）

| 方法 | 路径 | 说明 | 请求 |
|---|---|---|---|
| POST | `/api/kb/upload` | 上传文档 | form-data: file（pdf/docx/txt/md） |
| GET | `/api/kb/list` | 文档列表 | 无参数 |
| DELETE | `/api/kb/{id}` | 删除文档 | 路径传 id |

上传返回示例：
```json
{"doc_id": 1, "chunks": 82}
```

列表返回示例：
```json
[{"id":1, "title":"xxx.pdf", "type":"pdf", "status":"ready", "created_at":"2026-06-16T01:27:23"}]
```

### 智能问答模块 /api/chat（需鉴权）

| 方法 | 路径 | 说明 | 请求 |
|---|---|---|---|
| POST | `/api/chat/query` | 文字+图片问答 | form-data: question（必填）+ image（可选，传图片文件） |

返回示例：
```json
{
  "answer": "AI 生成的答案...",
  "image_observation": "图片描述（无图片时为空字符串）",
  "sources": [
    {"title": "来源文档名", "snippet": "相关片段前120字..."},
    ...
  ]
}
```

### 工单模块 /api/ticket（需鉴权）

| 方法 | 路径 | 说明 | 请求 |
|---|---|---|---|
| POST | `/api/ticket` | 创建工单 | `{"device":"设备名", "fault":"故障描述"}` |
| GET | `/api/ticket` | 工单列表 | 无参数 |
| GET | `/api/ticket/{id}` | 工单详情 | 路径传 id |
| PATCH | `/api/ticket/{id}` | 更新状态 | `{"status":"doing"}` （可选值: open/doing/done） |

创建返回示例：
```json
{
  "id": 1,
  "steps": {
    "raw": "AI 自动生成的四段检修步骤（风险预检/工具准备/检修步骤/验收标准）"
  }
}
```

详情返回示例：
```json
{
  "id": 1,
  "device": "空压机-XG200",
  "fault": "运行时噪音大",
  "steps": {"raw": "..."},
  "status": "open",
  "created_at": "2026-06-16T02:38:43"
}
```

## 三、前端调用示例

### axios 封装

```js
import axios from 'axios'

const api = axios.create({
  baseURL: 'http://127.0.0.1:8000',
  headers: { 'Content-Type': 'application/json' },
})

// 自动注入 token
api.interceptors.request.use(config => {
  const token = localStorage.getItem('token')
  if (token) {
    config.headers.Authorization = `Bearer ${token}`
  }
  return config
})

export default api
```

### 登录页调用

```js
// 登录
const res = await api.post('/api/auth/login', {
  username: 'admin',
  password: '123456'
})
localStorage.setItem('token', res.data.access_token)

// 拿到 token 后跳转主页
```

### 知识库页调用

```js
// 上传 PDF
const form = new FormData()
form.append('file', file)  // file 是 <input type="file"> 选的文件对象
await api.post('/api/kb/upload', form, {
  headers: { 'Content-Type': 'multipart/form-data' }
})

// 列表
const { data } = await api.get('/api/kb/list')

// 删除
await api.delete(`/api/kb/${docId}`)
```

### 问答页调用

```js
// 纯文字问答
const form = new FormData()
form.append('question', '压缩机异响怎么排查')
const { data } = await api.post('/api/chat/query', form, {
  headers: { 'Content-Type': 'multipart/form-data' }
})
// data.answer → 显示在聊天框
// data.sources → 显示引用来源

// 图片问答
form.append('image', file)  // 用户上传的设备照片
const { data } = await api.post('/api/chat/query', form, {
  headers: { 'Content-Type': 'multipart/form-data' }
})
```

### 工单页调用

```js
// 创建工单
const { data } = await api.post('/api/ticket', {
  device: '空压机',
  fault: '异响'
})

// 列表
const { data } = await api.get('/api/ticket')

// 详情
const { data } = await api.get(`/api/ticket/${ticketId}`)

// 更新状态
await api.patch(`/api/ticket/${ticketId}`, { status: 'doing' })
```

## 四、常见问题

### 401 Unauthorized
没登录或 token 过期。重新调 `/api/auth/login` 拿新 token。

### 422 Validation Error
请求体格式不对。参考上面各接口的请求体格式。

### 500 Internal Server Error
后端报错，把后台 uvicorn 窗口的红色 traceback 截图发我。
